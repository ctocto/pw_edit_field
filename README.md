# [pw_edit_field](https://github.com/lisen87/pw_edit_field)

flutter 密码输入框

flutter password input box

> Supported  Platforms
> * Android
> * iOS

## How to Use

```yaml
# add this line to your dependencies
pw_edit_field: ^1.0.0
```

```dart
import 'package:pw_edit_field/PasswordWidget.dart';
```

```dart
  PasswordWidget(
    textFieldCount: 6,
    onInputDone: (pw){
      /// pw Your password
      
    },
  ),
```
